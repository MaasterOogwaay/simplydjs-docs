# Introduction

Welcome to **simply-djs**. This Package simplifies many things so you can **Simply Develop**

:::danger WARNING

You need to know atleast the basics of JS and Discord.js

:::

## Installation

Get started by **installing the package**.

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs
  defaultValue="npm"
  values= {[

  ]

</Tabs>

## Using discord.js v12 ?

:::
