# Slash Support

:::danger WARNING

You need to have experience with making slash commands

:::

## Functions that support Slash commands
### General
- calculator
- embedPages
- menuPages
- rankCard

### Fun
- tictactoe
- rps

### Systems
- suggestSystem
- embedCreate
- betterBtnRole

:::danger WARNING

You need to deferReply before the function

:::

:::tip TIP
### The functions above shown has option called slash

Need to pass a [Boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) to it (Default: false).

### Example
```js
// interactionCreate 

// interaction.deferReply()
simplydjs.rps(interaction, {
  slash: true,
  // other options
})
```
:::

## Slash command Options
Refer the functions in Docs to see what options needed for that slash command
