# Slash Support

:::danger WARNING

You need to have experience with making slash commands

:::

## Functions that support Slash commands
### General
- calculator
- embedPages
- menuPages
- rankCard
- embedCreate
- stealSticker

### Fun
- tictactoe
- rps

### Systems
- suggestSystem
- ticketSystem
- embedCreate
- giveawaySystem
- embedCreate
- betterBtnRole
- btnrole

:::danger WARNING

You need to await the deferReply before the function

:::

:::info INFO

The package supports `Auto Slash Recognition !` So no more slash options.

:::

## Slash Options
Refer the functions in Docs to see what options needed for that slash command
